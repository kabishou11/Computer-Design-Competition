## Course Introduction

This repository contains materials for the AI Programming Fundamentals Course offered by Datawhale. The course covers practical AI programming skills, including machine learning, deep learning, and large language model applications.

## Course Content

| Module | Topic | Duration |
|--------|-------|----------|
| 1 | Python Basics | 2 weeks |
| 2 | Data Analysis | 2 weeks |
| 3 | Machine Learning | 3 weeks |
| 4 | Deep Learning | 3 weeks |
| 5 | LLM Applications | 2 weeks |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/datawhalechina/hello-agents.git
cd hello-agents

# Install dependencies
pip install -r requirements.txt
```
